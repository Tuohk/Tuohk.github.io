Vue文档小记之实例:upside_down_face:

声明式渲染

v-bind arrtribute被称为指令。指令前带有前缀v-。

指令意思：将这个元素的节点的attribute和Vue实例 property保持一致

组件化应用构建

父传子用prop组件

数据与方法

// 我们的数据对象
var data = { a: 1 }

// 该对象被加入到一个 Vue 实例中
var vm = new Vue({
  data: data
})

// 获得这个实例上的 property
// 返回源数据中对应的字段
vm.a == data.a // => true

// 设置 property 也会影响到原始数据
vm.a = 2
data.a // => 2

// ……反之亦然
data.a = 3
vm.a // => 3

这些数据改变时，视图也会被重新渲染，只有当实例被创建时已经存在于data中的property才是响应式。

vm.b = 'hi' //此时data没有b属性，无法渲染

使用Object.freeze()可阻止修改现有的property
